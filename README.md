# Enosh is good!

Everyone should know about enosh

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install enoshkes.

```bash
pip install enosh
```

## Usage

```python
import enosh

# returns 'coolest'
enosh.cool_maker('word')
```

## Example in js

```javascript
import { enoshifyer } from enosh

const coolest = enoshifyer('word')
```
